## Role-Specific Onboarding

### For IT Admin

**Key Responsibilities:**
- Technical infrastructure
- System maintenance
- API/integration management
- Technical support to vendors

**They Can Do:**
- Approve vendors (KYC)
- Approve products
- Handle orders
- Manage escrow
- Support tickets
- Create promotions
- View analytics

**They Cannot Do:**
- Process payouts
- Manage admin users

**Common First Tasks:**
1. Review pending vendor KYC
2. Approve/reject new products
3. Check system health in analytics
4. Monitor support tickets

---

### For Finance Admin

**Key Responsibilities:**
- Vendor payment processing
- Financial reconciliation
- Revenue tracking
- Transaction monitoring

**They Can Do:**
- Approve vendor payouts
- View all transactions
- Generate financial reports
- Monitor revenue

**They Cannot Do:**
- Approve vendors/products
- Manage orders
- Create promotions
- Manage admin users

**Common First Tasks:**
1. Review pending payout requests
2. Verify vendor payment details
3. Check financial analytics
4. Process approved payouts

---

### For Vendor Manager

**Key Responsibilities:**
- Vendor registration & verification
- Product quality control
- Vendor performance monitoring
- Dispute handling

**They Can Do:**
- Approve vendor KYC
- Approve/reject products
- Manage orders
- View analytics

**They Cannot Do:**
- Process payouts
- Manage escrow
- Create promotions
- Manage admin users

**Common First Tasks:**
1. Review pending vendor applications
2. Request vendor document updates if needed
3. Review new product listings
4. Approve quality products

---

### For Marketing & Support

**Key Responsibilities:**
- Customer support
- Promotional campaigns
- Customer relations
- Issue resolution

**They Can Do:**
- Handle support tickets
- Respond to customers
- Create promotions
- View analytics

**They Cannot Do:**
- Approve vendors/products
- Process payouts
- Manage orders
- Manage admin users

**Common First Tasks:**
1. Review open support tickets
2. Respond to customer issues
3. Plan promotional campaigns
4. Create attractive promotional banners

## Access Verification Checklist

After onboarding, verify:

- [ ] Team member can login to `/admin`
- [ ] They see only their role-specific tabs
- [ ] They can access their dashboard features
- [ ] They understand their responsibilities
- [ ] They have completed initial tasks
- [ ] They know how to get help/support
